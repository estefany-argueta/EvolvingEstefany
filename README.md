# EvolvingEstefany

Personal website for Estefany Argueta. This repo contains a simple static site (HTML + CSS).

How to publish via GitHub Pages:

1. Push this repository to GitHub.
2. In the repository Settings -> Pages, set the source to the `main` (or `gh-pages`) branch and the root `/` folder.
3. Wait a few minutes and visit `https://<your-username>.github.io/<repo-name>/`.

Local preview:

```bash
# from the repo root
python3 -m http.server 8000
# then open http://localhost:8000
```
